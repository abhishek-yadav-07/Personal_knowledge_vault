#Personal_Knowledge_vault
This is a simple terminal-based notes application created for the SOSC Recruitment Round 2.
It allows users to create, view, search, edit, and delete notes stored locally in a TXT file.


Features:

Create notes

View all notes

Search notes by keyword

Edit existing notes

Delete notes

Stores all notes in notes.txt

Runs entirely in the terminal



How It Works

Each note is saved in this format inside notes.txt:
TITLE: Note Title
CONTENT:
Note content can be multiple lines.


How to Run

1. Make sure Python is installed.

2. Open terminal in the project folder.

3. Run the program

4. Use the menu to manage your notes.




Files Included
